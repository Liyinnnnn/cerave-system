This README file provides a step-by-step guideline for examiners
to view the system demonstration and to access the hosted system.
The CeraVe Smart Skincare Advisor System supports three user roles:
- Administrator
- Consultant
- Consumer

The system provides comprehensive skincare guidance through AI-powered
chatbot (Dr. C), product catalog, appointment booking, and consultation
services.

3.3 Manage Products
------------------------------------------------------------

1. Click "Products" > "Add Product".

7. TROUBLESHOOTING
============================================================

------------------------------------------------------------
Q: I forgot my password. What should I do?
------------------------------------------------------------

A: 
1. Click "Forgot Password" on login page
2. Enter your email address
3. Click "Send Reset Link"
4. Check your email for password reset link
5. Click link and set new password
6. Login with new password

------------------------------------------------------------
Q: Email verification link expired. What should I do?
------------------------------------------------------------

A:
1. Click "Resend Verification Email" 
2. A new verification link is sent
3. Check email and click verification link
4. Account is now verified

------------------------------------------------------------
Q: I cannot see product recommendations from Dr. C.
------------------------------------------------------------

A:
1. Refresh the page
2. Check browser console for errors
3. Try a different browser
4. Clear browser cache and cookies
5. Try again

------------------------------------------------------------
Q: Appointment booking is not working.
------------------------------------------------------------

A:
1. Refresh the page
2. Check if you are logged in
3. Select consultant and valid date/time
4. Check internet connection
5. Try again

------------------------------------------------------------
Q: I did not receive consultation report.
------------------------------------------------------------

A:
1. Check spam/junk email folder
2. Wait 24 hours for report generation
3. Check "My Reports" section in account
4. Contact administrator if not received
